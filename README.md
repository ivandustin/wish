# wishwish

A general installer builder

## Usage

```sh
wishwish path/to/app -n <app name> -m <vendor name> -v 1.0.0
```

## Supported OS

Only Windows is supported at the moment.

## Dependencies

A Wix Toolset must be accessible in the `PATH` environment variable.
